# What is Linux?
- Linux is an open-source operating system widely used in servers, cloud computing, DevOps, and Site Reliability Engineering (SRE).
- It helps users interact with hardware and run applications efficiently.

# Why Linux is Important for DevOps/SRE
- Most servers run on Linux
- Cloud platforms use Linux extensively
- Kubernetes and Docker commonly run on Linux
- Useful for automation and troubleshooting
- Strong command-line support
  
# What is Linux Kernel?
- The Linux Kernel is the core part of the Linux operating system.
- It acts as a bridge between hardware and software by managing:
  * CPU
  * Memory
  * Devices
  * Processes

# What is a Shell?
- A Shell is a command interpreter that allows users to interact with the operating system using commands.
Examples:
* Bash
* Zsh
* Sh
  
# What is CLI?
- CLI (Command Line Interface) allows users to interact with the operating system using text-based commands instead of graphical interfaces.

# What is a Terminal Emulator?
- A Terminal Emulator is an application that provides access to the command-line interface.
Examples:
* GNOME Terminal
* Windows Terminal
* PuTTY

# What is a Prompt in Linux?
- A prompt is the symbol or text shown in terminal indicating that the shell is ready to accept commands.
Example:
user@server:~$

# Basic Linux Commands
**uname**
Displays operating system information.

**hostname**
Displays system hostname.

**whoami**
Displays current logged-in user.

**pwd**
Displays current working directory.
